# az_proof_initial_api

API para acompanhamento de pedidos e valores recebidos.

## Tecnologias
- Node.js
- Express
- MongoDB
- Mongoose

## Instalação
```sh
https://github.com/RobsonPMartins/az_proof_initial_api.git
```
## Entrar na pasta
```
cd az_proof_initial_api
```
## Intale as dependências
```
npm install
```
